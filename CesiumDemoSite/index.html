Clear Science Inc.

Object Float Demo

This is a demo of a Clear Science Inc. project designed to visually display an object's predicted float path based on data modeling algorithms using the Cesium viewer. This uses only controlled data to show what future functionality of this application will be able to do. As this project develops, more data will be made available for the user. This application is being developed as part of an internshp project by Jeff Janda and Veronica Washington.
